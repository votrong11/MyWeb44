Ứng Dụng Phân Tích Cấu Trúc URL

URL Mẫu Phân Tích:

https://example.com/products/list?page=2#top

Các thành phần chi tiết của URL:

Thành phần (Component) Giá trị cụ thể (Value) Ý nghĩa (Description)
Protocol (Giao thức) https Quy định cách thức truyền tải dữ liệu an toàn giữa trình duyệt và server.
Domain (Tên miền) example.com Địa chỉ định danh của website trên mạng Internet.
Path (Đường dẫn) /products/list Dẫn đến tài nguyên hoặc trang cụ thể trên server.
Query (Truy vấn) page=2 Chứa tham số gửi lên server để lọc hoặc phân trang dữ liệu (bắt đầu bằng dấu ?).
Fragment (Phân đoạn) top Định vị một vị trí/neo cụ thể trong trang HTML (bắt đầu bằng dấu #).
